Introducing the ZDK!

The best way to make applications for the Kinect

FEATURES

Get Started!

Download the ZDK, and then install it with one click. No command-line wizardry required!


Once you have downloaded it, plug in your Kinect or Asus Xtion and run one of our sample applications. Whether you are making a game, an interactive display, an art piece, or something the world has never seen before, using the ZDK you can hit the ground running and start using your creativity in minutes.

Consistent API

Whether on Mac or Windows, browser or native, Kinect SDK or OpenNI, the Zigfu API is identical. Never worry again about how to marshal those pesky C++ objects into .NET or what order to put the bitmasks in for starting up the sensor correctly. The busy-work is all handled for you behind the scenes in the ZDK, freeing you to spend your time where you should be: making your actual application. We even calculate all the skeleton joint orientations for you if you use the Kinect SDK!

Performance

We have ensured that even on low-end computers, all our software will run in real-time. That means depth and image streams that do not skip frames, even in web browsers, and software that will not gobble your systems memory.

Any Platform You Like

With the ZDK, you can create applications for Mac and PC. On Windows, you have the option of using the new Microsoft Kinect SDK as the middleware, or you can use OpenNI and NITE, which work on both OS X and Windows. You can also create apps using several different development environments, and we support both native and web applications. Using the ZDK, your options are unlimited.

Kinect in the Browser

For the first time, all the data from the Kinect or Asus Xtion is now available in the browser using the free Zigfu browser plugin, enabling you to create motion-controlled online games and web-apps. The browser plugin is a free one-click install that works in all major browsers on Mac and Windows to make it as easy as possible for users to get to your content. You can use Flash, Unity3D, or JavaScript and HTML5 to create your application. The motion-controlled web is here!

Unity3D

Zigfu has supported motion app development in Unity3D for over a year now. Our latest bindings for Unity include WebPlayer support (using the Zigfu browser plugin) and support for the Microsoft Kinect SDK. We include several sample applications with our Unity package, so you can start development from one of several fully functional motion-controlled applications.

Flash

New in this version of the ZDK is support for Adobe Flash, both in the browser and in standalone applications. Go ahead and make your app Flash-y ... sorry, bad joke, but it is good software!

HTML5/JavaScript

Make your website motion-controlled with just a few lines of JavaScript and HTML5! With full access to the image and depth streams and skeleton joint positions and orientations, you can now make your application web-based. Even better, you can now be fully buzzword-compliant with your app: now you can have a location-based social media daily deals site that runs in the cloud with a natural user interface, to help leverage your community and user retention. Pointy-haired bosses rejoice!
